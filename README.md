# kotlin-spring-boot-ddd-sample

Samples of DDD using kotlin

- [Exposed](https://github.com/JetBrains/Exposed)

# Run Application

## customer
```
$ gradle customer:bootRun
```

## contacts
```
$ gradle contacts:bootRun
```

http://localhost:8080/swagger-ui.html

## contacts
